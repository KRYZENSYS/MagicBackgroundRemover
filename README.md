# MagicBackground Remover

Telegram bot — rasmlardan orqa fonni AI bilan avtomatik o'chiradi.

## Xususiyatlar
- AI orqa fon o'chirish (rembg)
- Avtomatik foydalanuvchi tizimi (SQLite)
- Kanalga majburiy obuna
- Premium tizimi (kunlik limit)
- Referral tizimi
- Admin panel
- Ko'p tilli (UZ/RU/EN)
- Statistika

## O'rnatish
```bash
pip install -r requirements.txt
```

## Sozlash
`config.py` da:
- `BOT_TOKEN` — @BotFather dan
- `ADMIN_IDS` — admin ID
- `REQUIRED_CHANNELS` — kanallar

## Ishga tushirish
```bash
python bot.py
```
